### ionic preset vue

`ionic`, `vue`, `typescript`, `class style component`

`router`, `babel`, `history model for router`

`linter`

> vue create -p victorwang0526/ionic-preset-vue my-project

### build dist first

> npm run build

### add ios and android platforms

> npx cap add ios

> npx cap open ios

> npx cap add android

> npx cap open android


### sync your app

> npx cap copy
